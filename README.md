# Выявление брака на конвейере

Первый учебный проект по компьютерному зрению.  
Цель — научиться находить дефекты (царапины, трещины, загрязнения) на фотографиях деталей, которые движутся по конвейеру.

## Что делает проект

- Загружает фотографии деталей
- Использует нейронную сеть для поиска дефектов
- Показывает, где именно на картинке проблема

## Технологии, которые необходимы

- Python 3.10+
- Jupyter Notebook
- PyTorch / TensorFlow + Keras
- OpenCV
- Matplotlib / Seaborn (для визуализации)

## Архитектура проекта

Проект построен по простому пайплайну: данные → предобработка → модель → вывод результатов.
