# Обнаружение дефектов на конвейере

Первый учебный проект по **компьютерному зрению**.  
Цель — научиться автоматически находить брак (царапины, трещины, загрязнения и т.д.) на фотографиях деталей с конвейера.

## О проекте
  
Пока проект в начале: простые модели, предобработку, работа с небольшим набором данных.

## Что уже сделано

- Загрузка и просмотр изображений
- Базовая предобработка (изменение размера, нормализация)
- Первые эксперименты с классификацией (дефект / нет дефекта)
- Проба простой CNN (на PyTorch / Keras)

## Используемые технологии

- Python 3.10+
- Jupyter Notebook
- PyTorch / TensorFlow + Keras
- OpenCV (для работы с картинками)
- Matplotlib / Seaborn (визуализация результатов)

## Архитектура проекта (простая схема)

Изображения → Предобработка (resize, augment) → Модель (CNN / Transfer Learning) → Предсказание (класс дефекта или bounding box) → Визуализация результата
